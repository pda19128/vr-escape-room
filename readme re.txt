# VR Escape Room – Time Lab

Εργασία σε περιβάλλον A-Frame.

## Σενάριο
Ο χρήστης βρίσκεται σε ένα εργαστήριο χρόνου.
Ενεργοποιεί έναν ενεργειακό κύβο, ξεκλειδώνει τον πίνακα ελέγχου
και ανοίγει την πύλη διαφυγής.

## Αλληλεπίδραση
- Click σε 3D αντικείμενα
- Ενεργοποίηση οντοτήτων
- Χρονική καθυστέρηση πριν το τελικό μήνυμα

## Τεχνολογίες
- HTML
- JavaScript
- A-Frame 1.4.2

## Εκτέλεση
Ανοίξτε το `index.html` σε σύγχρονο browser (Chrome / Firefox).
